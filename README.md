A napari plugin for padbound.

----------------------------------

## Installation

You can install `napari-padbound` via [pip]:

    pip install napari-padbound

To install the latest development version:

    pip install git+https://github.com/your-username/napari-padbound.git

## Usage

1. Open napari
2. Go to `Plugins > padbound`
3. The plugin widget will appear in the napari viewer
