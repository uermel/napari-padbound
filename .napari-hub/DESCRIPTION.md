napari-padbound integrates MIDI controllers into napari for hands-on image annotation workflows. Connect a USB MIDI controller and use physical pads, knobs, faders, and buttons to navigate slices, select labels, adjust brush size, zoom, and undo/redo &mdash; with real-time LED feedback showing your current label colors directly on the controller hardware.

## Supported Controllers

Works with any controller supported by [padbound](https://github.com/uermel/padbound):
AKAI APC mini MK2, AKAI LPD8 MK2, AKAI MPD218, PreSonus ATOM, Synido TempoPad P16, Behringer X-Touch Mini, and Xjam.

## Usage

1. Connect a MIDI controller via USB
2. Open napari and go to **Plugins > padbound**
3. Load a 3D image and a Labels layer
4. Use your controller to navigate and annotate
